1# Client-Server:

  • 1. Benutzer ruft in seinem Browser eine Website auf.

  • 2. Browser fragt beim Server nach der zugehörigen IP-Adresse.

  • 3. Der Server liefert dem Browser die gewünschte IP-Adresse.

  • 4. Der Browser des Benutzers startet nun eine Anfrage (REQUEST) an die IP-Adresse des gesuchten Servers.

  • 5. Die Anfrage wird über das Internet zum entsprechenden Server geleitet.

  • 6. Der Server liefert als Antwort (RESPONSE) die gewünschte Seite.

  • 7. Der Browser versucht die empfangene Seite auf dem Monitor darzustellen.
